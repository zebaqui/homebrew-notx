# homebrew-notx

Homebrew tap for [notx](https://github.com/zebaqui/notx-engine) — a lightweight event-streaming / notification engine.

## Install

```sh
brew tap zebaqui/notx
brew install notx
```

## Usage

Once installed, the `notx` binary will be available in your PATH.

```sh
notx version
```

## Requirements

- macOS (Apple Silicon or Intel)
- [Homebrew](https://brew.sh)
